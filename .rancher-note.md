# Rancher Desktop Configuration

This project uses **Rancher Desktop** instead of Docker Desktop for container management.

## Commands

All container operations use `nerdctl` (Rancher Desktop's CLI):

- `make db-up` ŌåÆ uses `nerdctl compose up -d surrealdb`
- `make db-down` ŌåÆ uses `nerdctl compose down`
- `make db-reset` ŌåÆ uses `nerdctl compose down -v && nerdctl compose up -d surrealdb`

## docker-compose.yml

The `docker-compose.yml` file is compatible with `nerdctl compose` - no changes needed.

## Verification

Check if Rancher Desktop is running:
```bash
nerdctl ps
```

If you see "Rancher Desktop is not running", start Rancher Desktop application.
